# AI Model Inventory Manager — Server (Express + MongoDB)

 Live API URL: https://ai-model-inventory-server.vercel.app

---
## Overview

This is the backend server for the AI Model Inventory Manager web application.
It handles model storage, CRUD operations, authentication verification, and purchase tracking using Express.js, MongoDB, and ImgBB API integration.

## Key Features

Complete REST API for AI model management (CRUD + Purchase tracking) <br>

MongoDB Atlas integration for secure data storage <br>

ImgBB API support for image uploads <br>

JWT / Firebase token verification (for private routes) <br>

Deployed on Vercel / Render for fast and scalable hosting
